1. We can use "+" sign:


For Example:

fun main() {
    
    var name :String= "Lana"
    var surname :String="Miera"
    
    print(name+surname)
   
}

Output: LanaMiera

2. We can use "$" sign:

For example:

fun main() {
    
    var name :String= "Lana"
    var surname :String="Miera"
    var c:String="$name $surname"
    
    
    println(c)
    }
   
Output: Lana  Miera
